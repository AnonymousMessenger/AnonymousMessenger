Одноранговый, частный, анонимный и безопасный мессенджер, работающий через Tor. Это также бесплатное программное обеспечение с открытым исходным кодом, которое позволяет пользователям изменять и распространять его в соответствии с условиями GNU General Public License v3.

    Особенности:
    *Без рекламы, без серверов и без трекеров.
    *Полностью бесплатное программное обеспечение с открытым исходным кодом - и всё через Tor.
    *Двойное тройное сквозное шифрование Диффи-Хеллмана
    *Полностью одноранговое соединение с использованием скрытых сервисов
    *Включает Tor и obfs4proxy, поэтому вам не нужно какое-либо другое приложение для подключения
    *Возможность использовать мосты Tor (meek_lite, obfs2, obfs3, obfs4, scramblesuite)
    *Криптографическая проверка личности
    *Исключительная сетевая безопасность
    *Хранение зашифрованных файлов на Android
    *Исчезающие сообщения по умолчанию
    *Безопасность экрана
    *Оба одноранговых узла должны добавить друг другу луковые адреса, чтобы иметь возможность общаться.
    *Прямые голосовые вызовы через Tor (альфа-функция)
    *Фотографии профиля
    *Текстовые сообщения
    *Голосовые сообщения
    *Медиа-сообщения без метаданных
    *Отправка необработанных файлов любого размера (100 ГБ +)
    *Зашифрованный блокнот
    *Скоро появятся новые функции

Как использовать: https://anonymousmessenger.ly/how-to-use.html

По состоянию на декабрь 2020 года работа всё еще продолжается, и, хотя приложение полностью функционально, оно пока доступно только для Android.
